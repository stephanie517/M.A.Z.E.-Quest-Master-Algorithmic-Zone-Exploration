About

Maze Quest Overview

M.A.Z.E. Quest lets players choose from BFS, A*, WFS, DFS, or Dijkstra’s Algorithm to guide a character through a maze. The game not only entertains but also educates players about these algorithms and how they find the optimal path through a maze. It highlights computer science concepts like heuristic search and pathfinding efficiency, offering an interactive experience and visual demos.

Game Instructions

In this game, you will select an algorithm to navigate through a maze. The goal is to reach the end point using the chosen algorithm’s approach. You'll see the pathfinding process step-by-step, giving you insight into how each algorithm functions. Your mission is to complete the maze as quickly as possible using the best algorithm of your choice!
